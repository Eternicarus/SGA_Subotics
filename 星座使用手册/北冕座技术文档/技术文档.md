### 使用说明
#### 电脑（上位机）连接
1.修改以太网地址为   
   192.168.66.*   
    [*取值除1,2]
2. 使用终端或（vscode)进行远程连接，指令为  
   ssh lyr@192.168.66.2  
   #ssh方式 用户名@树莓派ip地址  
   #遇到WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!  输入  
   ssh-keygen -R IP  
   清除远程主机的公钥（此处IP更改为树莓派 IP）
3. 连接成功后运行 AUVr2025.exe

### 手柄控制
![](joystick.jpg)
左摇杆 前后左右平移
右摇杆 左右转、上浮下沉
X B 爪子张合
Y A 小臂旋转
左右扳机 大臂上下
RB LB 爪箱张合
start 定深PID开关
十字键左右 针筒右抽左推
十字键上下 摄像头上下
左摇杆按下 水面保持
右摇杆按下 水底保持
